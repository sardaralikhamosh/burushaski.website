# Burushaski Keyboard with GIF Export

A web-based Burushaski keyboard application that allows typing in Burushaski script and exporting text as GIF/PNG images.

## Features
- Virtual Burushaski keyboard
- Real-time text rendering
- Export as PNG images
- Export as animated GIFs (planned)
- Customizable text styling
- Responsive design
